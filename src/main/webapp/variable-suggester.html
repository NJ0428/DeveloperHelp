변수 추천기

한글 단어 입력

추천 변수명

한글 단어를 입력하고 추천 버튼을 클릭하세요.

예제 단어들

사용자
user, member, customer, account
데이터
data, info, information, dataset
목록
list, array, items, collection
상태
status, state, condition, flag
이름
name, title, label, identifier
시간
time, date, timestamp, duration
파일
file, document, attachment, media
설정
config, setting, option, parameter

사용 팁

명명 규칙
• camelCase: userName, userList
• snake_case: user_name, user_list
• PascalCase: UserName, UserList
변수 접두사
• current-, selected-, updated-
• is-, has-, can- (불린값)
• get-, set-, create- (함수)
변수 접미사
• -List, -Array (배열)
• -Count, -Index (숫자)
• -Data, -Info (정보)
